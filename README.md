# ereserves-lib-helper
新版教参平台下载脚本

## 介绍
本脚本用于帮助用户从 [新版教参平台](https://ereserves.lib.tsinghua.edu.cn/) 批量下载电子资源。

如果已经安装了 [Tampermonkey](https://www.tampermonkey.net/) 或其他用户脚本管理器，可以直接 [点击这里安装脚本](https://github.com/hotwords123/ereserves-lib-helper/raw/gh-pages/ereserve-lib-helper.user.js)。

## 使用方法
1. 安装脚本后，访问 [新版教参平台](https://ereserves.lib.tsinghua.edu.cn/) 并登录。
2. 搜索想要下载的教参，点击“立即阅读”进入阅读器页面（URL 形如 `https://ereserves.lib.tsinghua.edu.cn/readkernel/ReadJPG/JPGJsNetPage/xxxxxx`）。
3. 在阅读器页面，双击顶部的教参标题，即可开始下载。下载过程中会显示进度，下载完成后脚本会自动保存 PDF 文件。

## 注意事项
- 本脚本只能下载教材平台内的电子资源，用户需确保自己有权限访问这些资源。
- 本脚本仅供学习和研究用途，请勿公开传播下载的资源，以免侵犯版权。
